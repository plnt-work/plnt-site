plnt is the runtime that turns a micro-agent workflow spec + a Kubernetes GPU
backend into a running service — with canary, smoke test, and rollback, all
driven by a durable Temporal saga.

It sits in the middle of a three-repo stack:

- **[google-business](https://github.com/plnt-work/google-business)** — reference
  end-user product (Google Business Profile copilot). Consumes the runtime.
- **[microagents](https://github.com/plnt-work/microagents)** — public
  registry of workflow recipes. Pull path speaks S3 and OCI.
- **plnt** (this) — the runtime. `WorkflowRun` CRD → Temporal saga → Helm deploy.

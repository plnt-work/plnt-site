## What plnt is

Plnt is a server you run on your own hardware. It hosts **one long-lived planner LLM** that owns
your identity and memory, plus a few **named resident specialists** (research-librarian, code-shepherd,
inbox-triage). When work arrives, the planner spawns **ephemeral micro-agents** in sandboxed compute
units, each with just two tools — `search()` and `execute()` — over your filesystem.

Memory is plain files: JSONL + markdown + a derived FAISS sidecar. No SQL, ever.
